# Archive

This folder holds scripts that are useful as historical reference but are not part of the normal Pack 500 operating workflow.

## What belongs here

- One-off scripts tied to a specific event or year.
- Exploratory cleanup helpers that were superseded by a stable workflow.
- Scratch inspection tools that are still worth keeping for future debugging.

## Current contents

- `historical/2025/graduation_campout/generate_emails_to_leaders.py`: one-time helper for the 2025 graduation campout signup.
- `historical/2025/wolf_awards/pivot_awards.py`: one-time helper for the 2025 Wolf awards packet.
- `helpers/fix_roster.py`: older manual roster repair helper; the progress report script now handles raw roster exports directly.
- `helpers/tmp_inspect.py`: ad hoc HTML inspection helper used while refining the scouting.org scraper.

If you are trying to run a current Pack workflow, go back to the repo root README and start with one of the active scripts listed there.
